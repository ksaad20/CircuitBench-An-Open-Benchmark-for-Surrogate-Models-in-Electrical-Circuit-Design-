# Analog Benchmark Dataset

## Overview

The Analog Benchmark Dataset contains analog electronic circuits used to evaluate artificial intelligence methods for prediction, optimization, parameter estimation, surrogate modelling, and circuit analysis.

## Objectives

- Benchmark analog circuit modelling
- Evaluate AI regression methods
- Compare optimization algorithms
- Support reproducible research

## Circuit Categories

- Amplifiers
- Filters
- Oscillators
- Voltage References
- Signal Conditioning
- Sensor Interfaces

## Supported Benchmark Tasks

- Performance Prediction
- Component Parameter Estimation
- Surrogate Modelling
- Sensitivity Analysis
- Circuit Optimization

## Inputs

- Circuit topology
- Component values
- Simulation parameters

## Outputs

- Electrical characteristics
- Performance metrics
- Simulation responses

## Simulator Support

- ngspice
- PySpice

## Evaluation Metrics

- MAE
- RMSE
- R²
- Runtime
