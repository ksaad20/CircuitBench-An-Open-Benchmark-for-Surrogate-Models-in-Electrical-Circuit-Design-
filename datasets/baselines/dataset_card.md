# Baseline Models Dataset

## Overview

This directory stores baseline benchmark datasets and reference outputs used to establish reproducible benchmark performance.

## Purpose

Provide official baseline results for CircuitBench.

## Included Models

- Linear Regression
- Random Forest
- XGBoost
- Support Vector Regression
- Neural Networks

## Stored Information

- Predictions
- Metrics
- Runtime
- Configuration

## Outputs

Reference benchmark results.
